# 🎮 이상한 틱택토 (Strange Tic-Tac-Toe)

기본 틱택토의 규칙을 무시하고 매 게임마다 랜덤한 규칙이 적용되는 혼란스러운 게임입니다!

## 🎯 게임 특징

### 승리 조건 변형 (16가지)
- **기본**: 3칸 가로/세로/대각선
- **4칸**: 가로 4칸, 세로 4칸, 대각선 4칸
- **모양**: 2x2 정사각형, ㄱ자, ㅗ자, 십자가
- **위치**: 모서리 4칸, 중앙 4칸, 모든 변
- **특수**: 체스 나이트 패턴, 체스 퀸 패턴
- **조건**: 짝수/홀수 칸만, 대칭 구조

### 턴 규칙 변형 (6가지)
- **일반**: 일반 1칸 놓기
- **더블**: 한 턴에 2칸 놓기
- **뒤집기**: 턴마다 상대 돌 뒤집기
- **이동**: 턴마다 돌 이동 가능
- **봉인**: 턴마다 랜덤 칸 봉인
- **주사위**: 주사위 굴려서 위치 결정

### 특수 칸 (5가지)
- 🟩 점수 2배
- 💨 돌이 사라짐
- ❌ 상대 돌 강제 제거
- 🔒 한 번만 사용 가능
- 🔀 양쪽 모두 차지 가능

### 특수 능력 (5가지) - 각 1회 사용
- 🗑️ 상대 돌 제거
- 📍 돌 2개 놓기
- 🔄 위치 바꾸기
- ⏭️ 상대 턴 스킵
- 🚫 무효화

## 🚀 게임 실행

```bash
python game.py
```

## 📝 게임 규칙

1. **플레이어**: O (플레이어) vs X (AI)
2. **게임판**: 3x3 격자
3. **시작**: 매 게임마다 랜덤 규칙 적용
4. **입력**: 행,열 형식 (예: 0,0)

## 🎮 예시

```
이상한 틱택토 게임 시작! 🎮

📋 이번 게임 규칙:
  승리 조건: square_2x2
  턴 규칙: double_place
  특수 칸: {(1, 1): SpecialCellType.DOUBLE_SCORE}

  0 1 2
0 . . .
1 . . .
2 . . .

🔵 당신의 턴 (O)
  점수: 0 | AI 점수: 0
이동 입력 (행,열) 또는 'q'(종료): 0,0
```

## 🎲 게임 규칙이 어떻게 선택되나요?

매 게임마다:
- **승리 조건**: 16가지 중 1가지 랜덤 선택
- **턴 규칙**: 6가지 중 1가지 랜덤 선택
- **특수 칸**: 30% 확률로 1~3개 추가
- **점수제**: 30% 확률로 활성화
- **특수 능력**: 기본 활성화
